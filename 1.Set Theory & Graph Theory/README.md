# Set Theory and Graph Theory - 集合论与图论

## Set Theory - 集合论

### 主要内容

#### 对象

- 集合
- 关系
- 函数
- 自然数
- 基数

#### 思想

- 以逻辑为基础、以集合为工具、表示和构造各种数学对象

#### 内容

- `集合的基本概念`：集合之间的关系、运算、恒等式
- `二元关系`：表示、性质、函数、等价关系、序关系
- `自然数`：皮亚诺系统、自然数的运算、性质
- `基数`：有序集与无穷集、基数的比较
- `序数`：良序、超限归纳法

## Graph Theory - 图论

### 主要内容

#### 对象

- 由顶点和边构成的图

#### 思想

- 以集合论为基础、以图为工具、为各种二元关系建立模型

#### 内容

- `图的基本概念`：连通性、矩阵表示、带权图
- `欧拉图`、`哈密顿图`：边和顶点的遍历
- `树`：表示层级组织的关系
- `平面图`：判定、表示、性质
- `图的着色`：各种调度问题的模型
- `独立集`、`支配集`、`覆盖集`、`匹配`：各种应用问题
